## Chuleta

Para montar de rápido un servidor web en Python en el puerto 8000:

```Shell
python -m SimpleHTTPServer
```

Para compilar un archivo SCSS a CSS

```Shell
sass scss/estilos.scss css/estilos.css --style compressed --sourcemap=none
```
