Ejemplo de $rootScope

Y la anotación de punto.

Este es un ejemplo de $rootScope que es una variable global y que podemos configurar desde un módulo app.run() para que esté disponible en todos los controladores de la página. Solo recuerda inyectar $rootScope en cada controlador:

app.run(function ($rootScope) {
    $rootScope.name = {'saludo': 'Este es una variable $rootScope'};
});

app.controller('rootCtrl', function ($scope, $rootScope){
    
});
